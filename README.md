# JurisFlow - Sistema de Gestão Jurídica

Aplicação web desenvolvida em Python e Streamlit para gestão de escritórios de advocacia.

## Funcionalidades
- Cadastro de Clientes e Processos
- Upload e organização automática de arquivos
- Agenda de Audiências e Prazos
- Diário Processual
